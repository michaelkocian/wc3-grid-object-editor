# WC3 Grid Object Editor

WC3 Grid Object Editor is a browser-based tool for editing Warcraft 3 objects. Instead of using the traditional World Editor, this tool allows you to edit objects in a grid-based interface, making the process more intuitive and efficient.

## Features
- Edit Warcraft 3 objects directly in your browser.
- Grid-based interface for easier object management.
- No need to install additional software.
- No uploads to the internet.

## Getting Started
Visit the [editor](https://michaelkocian.github.io/wc3-grid-object-editor/editor.html).

### Usage
1. Open the `editor.html` in your browser.
2. Export .w3q .w3h .w3a .w3d .w3b .w3t .w3u files from your map.
3. Load the files into the editor. 
4. Start editing Warcraft 3 objects using the grid interface.
